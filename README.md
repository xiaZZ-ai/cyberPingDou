# Cyber Pingdou

Cyber Pingdou 是一个基于浏览器的拼豆图案设计工具，用于绘制像素风格的 Perler Beads / 拼豆作品。项目提供以画布为核心的编辑体验，支持色卡选择、本地保存、图片导出和颜色用量统计。

当前版本为纯前端应用，核心编辑流程不依赖账号系统或后端服务。

## 功能

- 基于 Canvas 的拼豆网格编辑器
- 支持常用画布尺寸和自定义行列
- 支持上色、橡皮擦、拖拽绘制、撤销、重做和清空画布
- 支持数字标尺和可开关的 5x5 辅助线
- 内置多品牌拼豆色卡，并支持品牌和颜色搜索
- 支持通过 JSON 导入自定义色卡
- 支持浏览器本地草稿保存
- 支持 PNG 图片导出
- 支持项目 JSON 导入和导出
- 支持颜色用量统计

## 技术栈

- React
- TypeScript
- Vite
- Zustand
- HTML Canvas
- localStorage

## 本地运行

安装依赖：

```bash
npm install
```

启动开发服务：

```bash
npm run dev
```

构建生产版本：

```bash
npm run build
```

预览生产构建：

```bash
npm run preview
```

```text
  constants.ts           画布预设、色卡和常量

  data/
    pixm-palettes.json   色卡数据
```

## 项目数据

作品以可编辑数据保存，而不是只保存为图片。每个格子会记录对应的颜色 id，因此项目可以继续编辑、统计颜色数量，并导出为可复用的 JSON 文件。

PNG 导出会基于项目数据重新绘制生成，不依赖页面截图。

## 色卡导入

项目支持通过 JSON 文件导入自定义色卡。导入后的色卡会保存在浏览器本地数据中，后续可以继续使用。

## 项目范围
